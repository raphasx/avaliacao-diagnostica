## Descrição da atividade
Este repositório contém uma página web desenvolvida utilizando apenas HTML, com o objetivo de simular um currículo pessoal.

A página possui as seguintes seções:

- Cabeçalho com nome, apresentação e menu de navegação
- Sobre mim
- Dados de contato
- Hobbies
- Rotina semanal (tabela)
- Mídia (imagem e vídeo)
- Curiosidades
- Rodapé com link de navegação

## Tecnologias utilizadas
- HTML5
